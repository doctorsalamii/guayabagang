### Key Files:
- **🌐 Frontend**: `index.html` + `css/styles.css`
- **🖼️ Images**: 70+ files (JPEG/PNG)
- **🧠 Logic**: `js/gallery.js`
- **📜 Metadata**: Favicons & webmanifest

## 🚀 Setup Instructions
1. `git clone` this bad boy 📥
2. Throw your images in the `images` folder like loot in Rust 🎒
3. Open `index.html` and enjoy the view 🖥️

## ➕ Adding Images
To feed the gallery:
1. Drop new images in `/images` like hot loot drops 🔥
2. Update `imageFiles` in `gallery.js` - no cheat codes needed 🕹️
